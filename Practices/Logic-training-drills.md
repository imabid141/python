### Drill 3 Review (Your Confusion)

Code:

```python
count = 0

for i in range(3):
    count += 1

print(count)
```

Let’s simulate it.

### Start

```
count = 0
```

### Loop runs 3 times

Because:

```
range(3) → 0, 1, 2
```

That is **three iterations**.

### Iteration 1

```
count = 0 + 1
count = 1
```

### Iteration 2

```
count = 1 + 1
count = 2
```

### Iteration 3

```
count = 2 + 1
count = 3
```

Final output:

```
3
```

✔ So **C is correct**.
